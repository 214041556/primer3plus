Help About

Primer3Plus

Primer3Plus is the module which runs primer3 to pick primers.

Primer3Manager

Primer3Manager allows to manage selected primers and to save them.

Primer3Prefold

Primer3Prefold allows to fold a sequence before primer selection to exclude regions with a secondary structure.

Primer3ComareFiles

Primer3ComareFiles allows to compare files with each other and the default values to identify differences.

Primer3Statistics

Primer3Statistics prints statistics about primer3plus usage.
